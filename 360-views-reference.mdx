## Interface reference

### Date range filter

- Located near the top of the view and typically defaults to a range such as "Last 90 days".

- Controls the time window that data is pulled from.

- Common options include Last 7 days, Last 30 days, Last 90 days, and a custom range.

- The selected date range applies across bills, hearings, and news within the 360 View.

### State filter

- Lets you constrain results to a subset of states.

- Supports multi-select so you can analyze several states at once.

- Useful for regional analysis and reporting focused on specific jurisdictions.

### Activity vs sentiment tabs

- **Activity** tab shows volume over time and by category.

- **Sentiment** tab shows stance or tone for the underlying content.

### Export and sharing

- Export the 360 View as a PDF for offline review.

- Share a link to the 360 View with collaborators.

- Use exported views in presentations and reports.

## FAQs

<Columns cols="2">
  <Card title="How current is the data?" href="#faqs" icon="refresh-cw" horizontal="false">
    Data is updated in real time.
  </Card>

  <Card title="Can I edit a 360 View after creating it?" href="#faqs" icon="pen-line" horizontal="false">
    You cannot edit an existing 360 View. To refine it, create a new one with an updated prompt; previous versions remain saved.
  </Card>

  <Card title="How many 360 Views can I create?" href="#faqs" icon="infinity" horizontal="false">
    There is no limit on the number of 360 Views you can create. All are saved and visible in the sidebar.
  </Card>

  <Card title="Can I share a 360 View?" href="#faqs" icon="share-2" horizontal="false">
    Yes. You can share a 360 View by sending a link or exporting and sharing the PDF.
  </Card>

  <Card title="What states are included?" href="#faqs" icon="map" horizontal="false">
    All 50 states, Washington, D.C., and federal legislation are included.
  </Card>

  <Card title="How far back does the data go?" href="#faqs" icon="calendar" horizontal="false">
    You can access any available time period using the date range filter.
  </Card>
